# 04 Inventory Agent

Question: Why does article X have inventory differences?

Workflow:
1. Intent detection.
2. Permission check.
3. Fetch ERP transactions.
4. Fetch WMS movements.
5. Fetch tickets, claims and incidents.
6. Retrieve SOPs and contracts.
7. Reconcile timestamps, units, locations and statuses.
8. Analyze patterns.
9. Calculate financial impact.
10. Summarize root cause with confidence and citations.
11. Recommend actions.

No ERP/WMS writeback without explicit approval.
