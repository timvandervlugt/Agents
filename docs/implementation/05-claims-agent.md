# 05 Claims Agent

Purpose: prepare claim decisions and draft communication while keeping humans in control.

Workflow:
1. Claim request.
2. Permission check.
3. Fetch ticket data.
4. Fetch delivery/order data.
5. Retrieve contract rules.
6. Collect evidence.
7. Check eligibility.
8. Calculate liability.
9. Draft claim email.
10. Human review.
11. Approved send and status tracking.

AI drafts. Humans approve.
